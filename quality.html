Committed to Quality

Valstom Energy follows a structured quality control and documentation approach to ensure that all supplied materials and integrated packages comply with project specifications, international standards, and client requirements across valves, piping, mechanical equipment, and related engineered items.

Our quality focus is on traceability, inspection coordination, documentation control, and delivery of compliant, installation-ready equipment for energy and industrial projects. All quality activities are aligned with project ITPs, datasheets, and applicable international codes and standards.

Material Certification

All materials are supplied with certification and traceability aligned with project and client requirements.

Documentation may include (as applicable):

  • EN 10204 certification (2.1 / 2.2 / 3.1 / 3.2)
  • Mill Test Certificates (MTC)
  • Heat number traceability
  • Chemical and mechanical test reports
  • NACE / sour service compliance where specified
  • Raw material origin documentation

Material certification is reviewed prior to dispatch to ensure full compliance with project specifications and applicable standards.

Standards & Compliance

Products and materials are supplied in accordance with applicable international standards and project specifications relevant to each product category.

Valves & Actuation

  • API 6D / API 6A / API 598
  • API 607 / API 6FA
  • IEC / ISO actuator standards
  • ASME B16 series

Pipes, Fittings & Flanges

  • ASTM / ASME material standards
  • ASME B31.3 / B31.1 / ASME B16.5 / B16.9 / B16.11
  • API 5L (where applicable)
  • MSS / EN / ISO standards

Pumps

  • API 610
  • ISO 5199 / ISO 2858
  • API 682 (mechanical seals where applicable)

Wellhead & Drilling Equipment

  • API 6A
  • API 17D (where applicable)
  • API 7-1 / API 7-2
  • NACE MR0175 / ISO 15156

All products are supplied in accordance with project datasheets, specifications, and applicable codes

Inspection & Testing

Inspection and testing activities are arranged or coordinated in accordance with project ITPs, client requirements, and applicable standards.

Non-Destructive Examination (as specified by project)

  • PMI (Positive Material Identification)
  • RT (Radiographic Testing)
  • UT (Ultrasonic Testing)
  • PT (Dye Penetrant Testing)
  • MT (Magnetic Particle Testing)
  • IGC testing where required
  • ISO 15848 - Fugitive Emission Testing (where specified)

Testing may be conducted in accordance with API 598, API 6D, API 6A, ISO 5208, and other project-specific procedures. All testing and inspection activities are coordinated with approved manufacturers and third-party inspectors in line with project requirements.

Vendor Qualification

Manufacturers and suppliers are evaluated and selected based on:

  • Technical Capability
  • Quality management systems
  • Certifications and approvals
  • Past Project Experience / Reference
  • Compliance with international standards
  • Third-party inspection and certification can be coordinated through recognized agencies such as BV, TÜV, DNV, Lloyd’s, Velosi, or client-nominated inspectors.

Type approval, product certification, and project-specific approvals can be supported where required.

Quality Assurance & Documentation

Prior to dispatch, all documentation is reviewed to ensure completeness and compliance with project requirements.

Documentation packages may include:

  • Material test certificates
  • Inspection and test reports
  • NDT records
  • Compliance certificates
  • Data sheets and drawings
  • Traceability documentation
  • Packing and delivery records

Our objective is to ensure that all supplied materials are compliant, traceable, and ready for installation upon delivery.
